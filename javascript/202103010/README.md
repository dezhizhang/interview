### 数据类型
```
1 String:任意字符串
2 Number:任意的数字
3 Boolean:true/false
4 Undefined:undefined
5 null:null
```
### 引用类型
```
1,Object:任意对像
2,Function:一种特别的对像
3,Array:一种特别的对像
```
### null的判断
```
typeof null === 'object'
```
### typeof可以判断的类型
```
可以判断undefined/数值/字符串/布尔值/function
不能判断null和object,object array
```
